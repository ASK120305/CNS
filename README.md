# CNS (Computer Network Security) - Cryptography Implementation

This repository contains implementations of various cryptographic algorithms and security concepts for Computer Network Security coursework.

## Overview

This project includes Python implementations of fundamental cryptographic algorithms including:
- Vigenère Cipher (CNS-3.py)
- RSA Encryption/Decryption Algorithm (CNS-6.py)
- Other cryptographic implementations

## Files

- `CNS-3.py` - Vigenère Cipher implementation with encryption and decryption
- `CNS-6.py` - RSA algorithm implementation with key generation, encryption, and decryption
- `CNS-2/` - Additional cryptographic implementations

## Getting Started

### Prerequisites
- Python 3.x
- Basic understanding of cryptographic concepts

### Running the Programs

1. **Vigenère Cipher (CNS-3.py)**
   ```bash
   python CNS-3.py
   ```
   - Enter a key when prompted
   - Enter plaintext to encrypt
   - The program will show both encrypted and decrypted text

2. **RSA Algorithm (CNS-6.py)**
   ```bash
   python CNS-6.py
   ```
   - The program generates RSA key pairs automatically
   - Enter a numeric message to encrypt
   - View the encryption and decryption process

## Algorithms Implemented

### 1. Vigenère Cipher
- Classical polyalphabetic substitution cipher
- Uses a keyword to shift letters in the plaintext
- Includes both encryption and decryption functionality

### 2. RSA Algorithm
- Asymmetric cryptographic algorithm
- Key generation using prime numbers
- Public/private key pair implementation
- Encryption and decryption demonstration
